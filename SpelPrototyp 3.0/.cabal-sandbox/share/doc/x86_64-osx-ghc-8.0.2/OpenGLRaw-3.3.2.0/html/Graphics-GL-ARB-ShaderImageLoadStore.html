Is the ARB_shader_image_load_store extension supported?

gl_ARB_shader_image_load_store :: Bool #

Is the ARB_shader_image_load_store extension supported? Note that in the presence of multiple contexts with different capabilities, this might be wrong. Use glGetARBShaderImageLoadStore in those cases instead.
