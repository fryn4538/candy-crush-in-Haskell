glGetEXTShaderImageLoadStore :: MonadIO m => m Bool #

Is the EXT_shader_image_load_store extension supported?

gl_EXT_shader_image_load_store :: Bool #

Is the EXT_shader_image_load_store extension supported? Note that in the presence of multiple contexts with different capabilities, this might be wrong. Use glGetEXTShaderImageLoadStore in those cases instead.
