glGetSGIXFragmentLighting :: MonadIO m => m Bool #

Is the SGIX_fragment_lighting extension supported?

gl_SGIX_fragment_lighting :: Bool #

Is the SGIX_fragment_lighting extension supported? Note that in the presence of multiple contexts with different capabilities, this might be wrong. Use glGetSGIXFragmentLighting in those cases instead.
